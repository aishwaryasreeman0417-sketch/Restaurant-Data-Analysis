# 🍽️ Restaurant Data Analysis

> 📊 **Data Analysis Internship Project – Cognifyz Technologies**

## 📖 Project Overview

This project was completed as part of the **Cognifyz Technologies Data Analysis Internship**. It focuses on analyzing restaurant data using Python to uncover meaningful insights through Exploratory Data Analysis (EDA), statistical analysis, and data visualization.

The project covers restaurant cuisines, city-wise distribution, pricing patterns, customer ratings, online delivery services, restaurant chains, and geographic analysis.

---

## 🚀 Technologies Used

- 🐍 Python
- 📊 Pandas
- 🔢 NumPy
- 📈 Matplotlib
- 🎨 Seaborn
- 🗺️ Folium
- 📓 Jupyter Notebook

---

# 📂 Dataset Features

The dataset contains information such as:

- Restaurant Name
- City
- Cuisines
- Aggregate Rating
- Votes
- Price Range
- Online Delivery
- Table Booking
- Latitude & Longitude

# ✅ Levels Completed

## 📌 Level 1

### 🍽️ Top Cuisines
- Identified the top three cuisines.
- Calculated cuisine percentages.
- Visualized cuisine popularity.

### 🌆 City Analysis
- Identified the city with the highest number of restaurants.
- Calculated average restaurant ratings by city.

### 💰 Price Range Distribution
- Analyzed restaurant price categories.
- Created distribution charts.

### 🚚 Online Delivery
- Calculated online delivery percentage.
- Compared ratings of restaurants with and without online delivery.

---

## 📌 Level 2

### ⭐ Restaurant Ratings
- Analyzed aggregate rating distribution.
- Identified the most common rating range.
- Calculated average customer votes.

### 🍕 Cuisine Combination
- Identified common cuisine combinations.
- Compared cuisine combinations with ratings.

### 🗺️ Geographic Analysis
- Visualized restaurant locations using Latitude and Longitude.
- Identified restaurant clusters.

### 🏪 Restaurant Chains
- Identified restaurant chains.
- Compared ratings and popularity using customer votes.
