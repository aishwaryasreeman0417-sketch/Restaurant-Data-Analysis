# Restaurant Data Analysis

## Internship
Cognifyz Technologies – Data Analysis Internship

## Project Overview

This project analyzes restaurant data using Python. The analysis includes cuisine trends, city-wise restaurant distribution, restaurant ratings, pricing patterns, online delivery services, geographic visualization, and restaurant chain analysis.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium
- Jupyter Notebook

## Levels Completed

### Level 1
- Top Cuisines
- City Analysis
- Price Range Distribution
- Online Delivery

### Level 2
- Restaurant Ratings
- Cuisine Combination
- Geographic Analysis
- Restaurant Chains
